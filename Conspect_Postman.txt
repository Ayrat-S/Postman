Что это значит?
Это набор запросов.

Чтобы начать писать запросы:
Жмем Add a request либо правой кнопкой по коллекции и тоже Add a request, либо жмем на три точки коллекции и тоже Add a request.

Перед отправкой запроса всегда жать кнопку Save (либо Ctrl + S).
В созданном новом запросе через метод GET отправляем запрос по адресу http://162.55.220.72:5005/

162.55.220.72 - это адрес железа
Внутри железа запущен http сервер с операционной системой Linux
У Linux есть возможность работы со множеством портов

На одном железе 162.55.220.72
На разных портах 5005 5007
Могут быть запущены разные http сервера
Т.е. не обязательно на одном ip запускать только лишь один единственный http сервер программу.

http сервер это программа, которая запускается на сервере.
Получается, http сервер 162.55.220.72 слушает 5007 порт.
И когда мы обращаемся в 5007 порт с запросом слэш, у нас возвращается немного другой ответ (по сравнению с портом 5005)
