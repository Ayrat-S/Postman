 
   МЕТОДЫ ЗАПРОСОВ в Postman:

   POST - создание объекта и отправка данных на сервер.
   GET - получение данных с сервера.
   PUT - обновление объекта.
   DELETE - удаление объекта.


   СТАТУС КОДА HTTP:

   1xx - Информация       Информирование о статусе передачи данных.
   2xx - Успех            Запрос выполнен успешно.
   3xx - Перенаправление  Запрос может быть выполнен, но по другому URL.
   4xx - Ошибка клиента   Указывает на ошибку на стороне клиента, из-за которой невозможно отправить запрос.
   5xx - Ошибка сервера   Указывает на ошибку на стороне сервера, из-за которой он не может обработать запрос.


   URL - Uniform Resource Locator - электронный адрес местонахождения ресурса.

   API - Application Programming Interface - интерфейс программирования приложений - совокупность инструментов и функций в виде интерфейса для создания новых приложений, благодаря которому одна программа будет взаимодействовть с другой.
